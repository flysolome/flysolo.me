Lead -> Dutch proposal -> Rate check -> Scope -> Contract -> E-signature -> Moneybird invoice -> Payment

Use FlySolo to:

- Turn cloud briefs into structured Dutch proposals
- Explain pricing for architecture, advisory, and implementation work
- Make assumptions, dependencies, and exclusions explicit
- Prepare contract-ready terms before work starts
- Carry accepted work into Moneybird invoice and payment follow-up

Early beta onboarding starts with one real or anonymized cloud opportunity.
